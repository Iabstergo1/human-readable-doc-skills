# 第三章 模型设定

## 概念界定

本章把研究对象定义为个体在约束条件下的选择结果。模型只描述变量之间的
条件关系，不把未观测因素解释为已经验证的因果机制。

## 研究问题

研究问题是：在控制个体特征和地区固定差异后，核心解释变量是否与结果
变量存在稳定关系。该关系需要在后续实证章节中检验，本章只给出模型形式。

## 符号与变量定义

首先，令 \(y_i\) 表示个体 \(i\) 的结果变量。其次，令 \(x_i\) 表示
核心解释变量，令 \(z_i\) 表示控制变量向量。地区固定效应用
\(\mu_r\) 表示，误差项用 \(\varepsilon_i\) 表示。

## 基准模型

基准模型写作：

```math
y_i = \alpha + \beta x_i + \gamma z_i + \mu_r + \varepsilon_i
```

其中，\(\beta\) 是本文关注的参数。若 \(\beta > 0\)，说明在控制变量
给定时，\(x_i\) 的上升与 \(y_i\) 的上升相关。该解释仍需依赖识别假设
和稳健性检验，不能单独推出因果结论。

## 识别假设

模型解释依赖三个条件：

- 控制变量覆盖主要可观测差异。
- 地区固定效应吸收稳定的地区层面差异。
- 误差项中不存在与 \(x_i\) 系统相关的遗漏因素。

如果以上条件不成立，估计结果可能包含遗漏变量偏误。需要在实证章节通过
替代变量、分组检验或工具变量策略进一步讨论。[citation needed]

## 局限

本章没有提供数据来源、样本选择规则和变量构造细节。这些内容需要在数据
章节补充，并与模型符号保持一致。
