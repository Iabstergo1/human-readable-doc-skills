# Writer Loop Style Learning

## Input

User asks: "Learn the style of this sample and write the next section in the
same voice. Do not reuse any facts from the sample."

Sample says: "The report uses short paragraphs, restrained transitions, and
specific operational nouns. It names constraints before recommendations."

## Expected Route

Use document workflow, then style learning mode.

## Expected References

- `references/01-writing-loop.md`
- `references/02-human-readable-style.md`
- `references/10-quality-gates.md`
- `references/13-style-distillation.md`

## Expected Lint Categories

- `style-content-leak` if sample facts or names are copied into the target.

## Expected Behavior

Build a session-only style pack. Extract rhythm, paragraph shape, abstraction
level, heading habits, terminology habits, and reader relationship. Do not
create durable storage unless the user explicitly asks.
