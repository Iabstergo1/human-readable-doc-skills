# 平台劳动研究笔记

## 概念界定

本文中的“算法管理”指平台通过派单、计价、评分和惩罚机制影响劳动者
行为的过程。该定义需要与一般意义上的数字化管理区分开来，因为后者
不一定包含实时评价和自动化约束。

## 论证线索

如果平台规则决定了劳动者的收入波动，那么研究重点不应只放在个体选择
上，还需要分析规则透明度、申诉机制和数据可见性。[citation needed]
