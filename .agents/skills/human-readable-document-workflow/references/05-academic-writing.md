# Academic Writing

Academic writing should make concepts, evidence, and argument structure explicit.

## Rules

- Define key concepts before using them in an argument.
- Use consistent variable names and notation throughout the document.
- Keep each paragraph anchored by a clear topic sentence.
- Separate assumptions, definitions, claims, evidence, and implications.
- Use citation placeholders when sources are missing.
- Never invent papers, authors, years, journals, DOIs, or quotations.
- If no source is available, write `[citation needed]` or ask the user for source material.

## Argument Chain

For each main claim, check:

1. What concept or variable is involved?
2. What assumption is required?
3. What evidence supports the claim?
4. What limitation or countercase matters?
5. What conclusion follows without overclaiming?

## Citation Placeholders

Use placeholders that are easy to replace:

```markdown
According to prior work on [topic] [citation needed], ...
```

Do not fabricate a bibliography to make a draft look complete.
